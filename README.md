# pm-distill · PM 工作经验提炼 Skill

一个给 B端 PM 用的 Claude Code Skill，专门用于把每天的工作对话提炼成可携带、可复用的经验条目。

适用背景：在公司电脑无法使用个人云服务的情况下，通过 Claude Code 做工作经验的「即时结晶」。

---

## 包含模板

| 模板 | 适用场景 | 字数限制 |
|------|----------|----------|
| 模板 A · 标准提炼 | 需求分析、PRD、业务判断、方案设计 | 200字以内 |
| 模板 B · AI经验提炼 | 用 Claude Code 完成某个 AI 任务后 | 150字以内 |
| 模板 C · 最小提炼 | 很忙，只想快速记一句话 | 20字以内 |

---

## 安装方法（公司电脑，无需 Git）

### 第一步：下载

1. 打开本页面，点击右上角绿色按钮 **`Code`**
2. 点击 **`Download ZIP`**
3. 下载完成后，解压缩

### 第二步：放置文件

把解压后的 `pm-distill` 文件夹（里面有 `SKILL.md`），放到你的 Claude Code 工作目录下：

```
你的工作目录/
└── .claude/
    └── skills/
        └── pm-distill/       ← 把这个文件夹放在这里
            └── SKILL.md
```

> 如果 `.claude/skills/` 文件夹不存在，手动新建即可。

### 第三步：验证

在工作目录打开 Claude Code，输入：

```
帮我提炼今天的工作经验
```

如果 Claude 开始按模板提炼，说明安装成功。

## 使用方式

直接用自然语言触发，不需要记命令：

- `"帮我提炼一下"` → 标准提炼
- `"今天的复盘"` → 标准提炼  
- `"这次AI用得不错，记录一下"` → AI经验提炼
- `"没时间，给我一句话"` → 最小提炼
- `"distill"` → 标准提炼（最短触发词）

## 配套：个人知识归档（回家用）

每周把这周提炼出来的内容，用以下 Prompt 喂给个人电脑上的 Claude，建立持续迭代的知识库：

```
你是我的个人产品经验库管理者。
我是一个供应链领域的B端PM，有约10年经验，正在向AI方向转型。

我会不定期给你输入这周的工作经验片段，你需要帮我：
1. 归入合适的分类（需求分析/业务判断/供应链领域/AI应用/个人方法论）
2. 提炼成可复用的「经验条目」，每条50-100字
3. 如果和之前的条目有关联或矛盾，指出来
4. 输出完整的最新知识库

当前知识库内容：
[把上周的归档结果粘贴在这里，第一次留空]

本周新增片段：
[把这周记住的内容输入在这里]
```

把 Claude 输出的结果保存到本地 txt 文件，每周覆盖更新。
